# AI Spider

Discover and summarize job listings from across the web using AI.

## Quick Start

```bash
uv sync
uv run main.py
```

## Features

- Fetches job listings from the web
- AI-powered summaries via LangChain + Ollama

## Requirements

- Python 3.14+
- Ollama with llama3.2:3b model

## Tech Stack

- Python 3.14+ | LangChain | BeautifulSoup4 | requests
